# N0torius

N0torius es un sistema de gestión de notas para el ambito académico. Nos permite crear : cursos, asignaturas y examenes . Aparte los examenes son editables y pueden ser borrados

|------------|---------|
|Página	     |    7    |
|Servicios   |    5    |
|Modelos     |    3    |
